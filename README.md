# ogle

`ogle` is a Python-based toolkit designed to streamline the handling of data from the Optical Gravitational Lensing Experiment (OGLE). It provides a structured environment for managing OGLE-specific data types, automating common workflows, and integrating survey results into broader astrophysical research pipelines.

The `ogle` package simplifies this by offering:
- Specialized Data Handling: Native support for OGLE photometry and light curve formats.
- Integrated Tooling: Pre-built functions for data cleaning and time-series analysis.
- Research Ready: Designed to work seamlessly with the standard Python science stack (NumPy, Astropy, etc.).
